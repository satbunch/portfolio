プログラミング教室のLP

(https://proud-saiki-9477.main.jp/satomi_web_check/)

普段は社会人向けにパソコン教室を運営している「ＡＢＣパソコンスクール」様が、夏休みのイベントとして小学生向けに電子クラフト・プログラミング教室を行う。集客のためのＬＰを作成してほしいと依頼を受けた。

  • 制作期間

    １０日
    (企画書：２日、デザイン：２日、コーディング：６日)

  • 使用技術

    HTML / CSS / jQuery
    AdobeXD / photoshop / Canva

Comment

● サイトについて

クライアントからの要望：
夏のイベントとして集客をして今後の受講生の拡大を目指したい。イベントの日時と申し込み方法がはっきりとわかるようにしたい。

ターゲット：
小学生の子どもを持つ３０～４０代の親。プログラミングは難しそうだが、学ぶことで将来の仕事の幅が広がることを期待している。

● 工夫した点

  • －子どもの画像を多く配置し、体験性の訴求
  • －登録をメールアドレスだけにする
  • －色や背景で難しくないイメージを持たせる
  • －体験者の声を追加で提案し、安心感と期待感を与える
  • －「限定●名」「×月×日まで」等の限定性の訴求
  • －文字を大きくして老眼の方も離脱しにくくする
